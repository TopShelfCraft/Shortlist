Installation

Shortlist is a standard Craft plugin. Installation is as simple as dropping the 'shortlist' folder into your Craft/plugins directory, and clicking 'Install'.

Craft Plugin Installation

  1. Upload the shortlist folder to your craft/plugins/ folder
  2. In your Craft control panel, got to Settings > Plugins. Click Install on the Shortlist plugin
  3. Optionally the Shortlist settings and update any values you might want to change.
  4. All done. View the tag's you'll need to add to your front-end templates in the Usage section.
